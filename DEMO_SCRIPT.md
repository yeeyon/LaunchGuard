# Three-minute LaunchGuard demo

- **0:00–0:20** — “Most deployment failures are configuration failures. LaunchGuard is the release review before production finds them.”
- **0:20–0:40** — Click **Run the broken-repo demo**. Explain that it needs no GitHub, OpenAI, or database credential.
- **0:40–1:10** — Show the completed deterministic report: the fixture is deliberately unsafe, and the result is evidence-backed rather than simulated progress.
- **1:10–1:45** — Point to the readiness score, critical/high distribution, and blockers. Explain the capped severity deductions.
- **1:45–2:20** — Open the missing build script finding. Show evidence, remediation, verification commands, and the reviewable unified diff.
- **2:20–2:40** — Download Markdown or open printable HTML. Highlight the ship checklist.
- **2:40–3:00** — Explain that GPT-5.6 is optional, receives redacted context, and adds cross-file reasoning while deterministic checks remain the scoring spine.
